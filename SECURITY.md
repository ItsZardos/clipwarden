# Security Policy

## Supported versions

I currently provide security fixes for:

- v1.0.x

Older versions may not get security patches.

## Reporting a vulnerability

If you find a security issue, email me at:

**ethan@ethantharp.dev**

Please do not open a public GitHub issue for vulnerabilities.

I aim to respond within 72 hours.

If your report leads to a fix, I am happy to credit you in the release notes with your permission.

## Scope

### In scope

- Detection bypasses
- False positive chains
- Dependency issues that impact security
- Log leakage

### Out of scope

- Attacks that require physical access to the machine
- Privilege escalation classes ClipWarden does not claim to prevent
- Windows platform bugs outside this project
